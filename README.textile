h1. mkp_if_en

This conditional tag examines the url of the current page and determines if the url begins with 'en.' This allows for a custom page to be rendered in English.

h2. Links

* "Github Code Repository":https://github.com/michaelkpate/mkp_if_en

h2. mkp_if_en

h3. Attributes

h4. None

h3. Examples

You will want something similar to this in your individual article template to display the alternative formatting.

bc. <txp:mkp_if_en>
 <txp:output_form form="layout_archives_english" />
<txp:else />
 <txp:output_form form="layout_archives_standard" />
</txp:_mkp_if_en>
